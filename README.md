- Unlimited learning and testing!
- Upload vocabulary list easily!
- Play a fun game to memorize vocabulary!

## Vocabulary List
Your vocabulary file type must be a .json file. 
You must fill the list out like this. 

```
[
    {
        "vocab": "",
        "definition": "",
        "sentence": ""
    },
    {
        "vocab": "",
        "definition": "",
        "sentence": ""
    }
]
```

This is the example list:

```
[
    {
        "vocab": "Ubiquitous",
        "definition": "Present, appearing, or found everywhere.",
        "sentence": "Mobile phones are now ubiquitous in our daily lives."
    },
    {
        "vocab": "Ephemeral",
        "definition": "Lasting for a very short time.",
        "sentence": "The beauty of a sunset is ephemeral."
    },
   {
        "vocab": "Pragmatic",
        "definition": "Dealing with things sensibly and realistically.",
        "sentence": "She took a pragmatic approach to solving the budget crisis."
    },
    {
        "vocab": "Meticulous",
        "definition": "Showing great attention to detail; very careful and precise.",
        "sentence": "The painter was meticulous about every brushstroke."
    },
    {
        "vocab": "Resilient",
        "definition": "Able to withstand or recover quickly from difficult conditions.",
        "sentence": "The community was resilient in the face of the disaster."
    }
]
```
